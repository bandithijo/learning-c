## Problem statement:

Write a program to add two numbers without using "+" operator operator.

## Idea

Ide is to use increment and decrement operators.

## Algorithm

1. Step 1 - x++; y--;
2. Step 2 - Repeat Step 1 until Y becomes 0

```
    x = 3, y = 4;
(1) x = 4, y = 3;
(2) x = 5, y = 2;
(3) x = 6, y = 1;
(4) x = 7, y = 0;
```
