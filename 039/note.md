# 39 for and while Loops

## While Loop

```
while(expression)
{
    statement1;
    statement2;
    .
    .
    .
}
```

```c
int i = 3;
while (i > 0)
{
    printf("%d", i);
    i--;
}
```

## For Loop

```
for (initialization; condition, increment/decrement)
{
    statement1;
    statement2;
    .
    .
    .
}
```

```c
for (int i = 3; i > 0; i--)
{
    printf("%d", i);
}
```
